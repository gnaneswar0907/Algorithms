### [FIRST UNIQUE CHARACTER](https://github.com/gnaneswar0907/Algorithms/blob/master/30DayMayLeetCode/Day_3/FirstUniqueCharacter.java)

s = "leetcode"
return 0.

s = "loveleetcode",
